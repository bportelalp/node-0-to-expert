# Aplicación de consola que imprime una tabla de multiplicar

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar      [número] [requerido]
  -l, --limit    Límite de tabla                          [número] [defecto: 10]
  -v, --verbose  Mostrar por consola                 [booleano] [defecto: false]
```
